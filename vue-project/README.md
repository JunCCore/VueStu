# vue-project

>  Vue.js project

## vue 项目搭建工作记录

``` bash
    #开发环境准备
    
    1.官网下载安装node.js
    2.安装依赖管理的包的工具，npm
        npm install
    3.安装淘宝镜像
        cnpm install
    4.安装vue的脚手架 百度
    
   #安装之后可查看是否安装成功
   
    node --version
    npm --version
    cnpm --version
    vue
    安装成功会有提示信息
    -cnpm run dev
# serve with hot reload at localhost:8080

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
